# proyecto-react 🚀

Proyecto desarrollado en React + Vite como parte del aprendizaje de componentes con props.

## 🛠 Tecnologías utilizadas
- React 18
- Vite
- React Router DOM

## 📁 Estructura del proyecto
```
src/
├── components/
│   ├── layout/
│   │   └── Navbar.jsx
│   ├── sections/
│   │   └── Hero.jsx
│   └── ui/
│       └── WhatsAppButton.jsx
├── CardPrefiles/
│   └── Prefiles.jsx
├── screens/
│   ├── Home.jsx
│   ├── Products.jsx
│   ├── Contact.jsx
│   └── PrefilesCard.jsx
└── App.jsx
```

## 📄 Páginas disponibles
| Ruta | Descripción |
|------|-------------|
| `/` | Inicio con Hero |
| `/products` | Productos |
| `/contact` | Contacto |
| `/profile` | Perfiles con cards |

## ▶️ Cómo correr el proyecto
```bash
npm install
npm run dev
```
